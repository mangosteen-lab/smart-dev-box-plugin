## 一、报告说明
### （一）分析范围
本次分析覆盖2024年秋季拍卖会、2025年春季拍卖会、2025年秋季拍卖会三大专场，拍品包含传统书画、西画、油画、版画，不含写经，无陶瓷、紫砂等其他品类数据。

### （二）数据依据
严格依据已上传的2024秋拍、2025春拍、2025秋拍书画专场分析报告，不篡改、不编造，无数据项明确标注"无相关数据"。

### （三）核心维度
围绕拍品规模、年代结构、估值价格、品类构成、名家价值、来源信息六大维度开展横向对比。

### （四）分析目标
梳理三场拍卖供给、价格、品类、名家、渠道特征，提炼共性规律与差异，总结市场趋势，汇总全场次数据质量问题。

## 四、三场共性规律
1. 供给结构稳定：近现代+清代为核心供给，占比超80%，高古作品持续稀缺。
2. 价格结构一致：均为金字塔型，低端走量、高端锚定，千万级拍品占比均不足0.4%。
3. 专场定位统一：以传统中国绘画、书法为绝对核心，西方艺术品类占比极低。
4. 名家格局固化：黄宾虹、吴昌硕、佚名为核心名家，市场认可度稳定。
5. 来源信息匮乏：无明确来源占比均超99%，仅西泠既往上拍有少量有效数据。
6. 数据问题雷同：均存在标注不规范、录入错位、信息缺失、分类模糊四大问题。

## 五、三场个性差异
1. 2024秋拍：总规模最大；无底价拍品数量最多；徐渭作品创三场最高价；张充和上拍量独居榜首。
2. 2025春拍：民国作品占比最高；传统绘画占比峰值；西方艺术品类大幅收缩；黄宾虹上拍量居首。
3. 2025秋拍：总规模最小；无无底价拍品；其他类年代占比最高；佚名上拍量第一；油画、西画供给回升。

## 六、市场发展趋势总结
1. 供给趋势：拍品总规模小幅收缩，供给持续向近现代、清代集中，高古作品供给进一步缩减。
2. 价格趋势：低端走量拍品占比扩大，千万级高端锚点拍品占比小幅提升，无底价拍品逐步退出。
3. 品类趋势：传统书画核心地位持续强化，其他类拍品精简，西方艺术品类供给波动调整。
4. 名家趋势：核心名家格局稳定，黄宾虹市场地位持续上升，佚名作品上拍量增加。
5. 渠道趋势：来源信息缺失率小幅下降，西泠既往上拍可追溯来源稳步增长，整体流传有序性缓慢提升。

## 七、全场次数据质量问题总览
### （一）标注不规范
1. 年代标注格式混乱，朝代、具体年份、年代区间混用。
2. 估价格式不统一，全角/半角冒号、前缀符号不一致。
3. 材质、形式标注碎片化，非标准术语大量出现。
4. 来源标注仅用"B"，无释义，格式不统一。

### （二）录入错误
1. 年代列混入尺寸、形式等非年代信息，列错位严重。
2. 估价高低价逻辑颠倒，年份标注明显不合理。
3. 重复列、合并单元格导致数据读取异常。

### （三）信息缺失
1. 来源信息缺失率超99%，备注栏利用率极低。
2. 作者、估价、尺寸、备注等关键信息部分缺失。

### （四）分类模糊
1. 跨年代描述（明末清初、清末民国）导致分类偏差。
2. 书画合璧、复合作品分类边界不清，归类歧义。
3. 材质、形式模糊标注导致分类困难，其他类扩容。
